스프라이트 이미지는 이미지를 여러 개 뭉쳐놓은 것을 의미한다.
이미지를 1개의 파일에 뭉쳐놓으면 웹페이지 요청 시간을 줄이고 쉽게 관리할 수 있다.
스프라이트 이미지를 수작업으로도 만들 수 있지만 일반적으로 프로그램을 사용한다.
CSS Sprites Generator
https://www.toptal.com/developers/css/sprite-generator

상단에 있는 옵션은 다음과 같이 지정하자.
* Padding between elements(px) : 0
* Align elements : Binary Tree

모든 설정을 완료하고 <GENERATE> 버튼을 누르면 스프라이트 이미지가 생성된다.
스프라이트를 생성하면 웹 페이지 바로 아래에 간단한 사용 방법이 나온다.

    <!-- 스프라이트 이미지 -->
    <style>
        #main_header > a.left {
            background: url('sprites.png');
            background-position: 0px 0px;
            text-indent: -99999px ;
        }
        #main_header > label.right {
            background: url('sprites.png');
            background-position: -62px 0px;
            text-indent: -99999px; 	/* 태그 내부에 있는 글자를 보이지 않게 만들고자 터무니 없는
				값을 적용하여 글자를 화면 밖으로 밀어버리는 코드이다. */
        }
    </style>

        <header id="main_header">
            <a class="left" href="#">Main</a>
            <h1>Mobile</h1>
            <label class="right" for="toggle" onclick="">Toggle</label>
        </header>

"text-indent 속성을 사용하지 말고, 글자를 입력하지 않아도 되잖아요?"

물론 그렇게 해도 된다. 하지만 HTML 코드를 나중에 살펴볼 때 해당 태그가 어떠한 태그인지 쉽게 알게 하고자 글자를 입력했다.
